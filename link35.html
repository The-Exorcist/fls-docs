Использование React

Стартовый шаблон "Чертоги Фрилансера" v3.0.0 поддерживает работу с популярной JavaScript библиотекой React.

Для того чтобы начать работу с React досточно раскомментировать строку
import Index from ‘./react/Index.jsx’; в файле src/js/app.js

Рабочие файлы находятся в папке src/js/react
