Модуль экранной прокрутки страницы (FullPage)

Подключение модуля

[JS] В файле js/app.js раскомментировать строку import ‘./libs/fullpage.js’

[SCSS] В файле src/scss/base.scss раскомментировать строку @import “base/fullpage”;

[HTML] Использовать сниппет fullpage или построить HTML-структуру самостоятельно:

Настройка модуля

Буллеты

Есть возможность добавить кнопки переключения секций (буллеты). Для этого к объекту с дата атрибутом data-fp добавить дата атрибут data-fp-bullets

Эффекты переключения

Для изменения эффекта переключения, объекту с дата атрибутом data-fp добавить дата атрибут data-fp-effect и как значение добавить один из следующих параметров fade, cards, slider (по умолчанию)
